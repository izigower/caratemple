# Guide utilisateur CaraTemple

## 1. Découvrir CaraTemple
- **Accueil** : affiche les discussions récentes, un menu latéral et les contenus mis en avant.
- **Navigation principale** : le bouton burger ouvre/ferme le menu sur mobile, les actions "Rejoindre", "Connexion" et "Créer un post" restent accessibles en permanence.
- **Accessibilité** : utilisez le lien "Aller au contenu principal" pour passer directement aux discussions.

## 2. Créer un compte
1. Cliquez sur "Rejoindre le Temple" puis remplissez le formulaire d'inscription.
2. Les champs sont validés en direct (longueur du mot de passe, format e-mail, accord des CGU).
3. Soumettez pour créer votre profil ; vous serez automatiquement connecté.

## 3. Se connecter / se déconnecter
- Cliquez sur "Connexion" et renseignez vos identifiants.
- Une fois connecté, votre avatar apparaît dans la barre d'application et les actions de publication deviennent disponibles.
- Utilisez le menu utilisateur (avatar) pour vous déconnecter en toute sécurité.

## 4. Participer aux discussions
- Depuis la page d'accueil, cliquez sur une discussion pour consulter le fil complet.
- Les compteurs indiquent le nombre de vues, de réponses et de likes.
- Pour répondre : utilisez le formulaire en bas du fil, prévisualisez puis envoyez.
- Vous pouvez modifier ou supprimer vos propres messages tant que la discussion reste ouverte.

## 5. Créer un nouveau sujet
1. Cliquez sur "Créer un post".
2. Saisissez un titre, choisissez une catégorie et décrivez votre sujet.
3. Ajoutez des tags optionnels pour faciliter la recherche.
4. Publiez : le nouveau sujet s'affiche instantanément dans le flux principal.

## 6. Likes et suivi
- Cliquez sur l'icône "cœur" pour aimer une réponse ; le compteur est mis à jour en direct après rafraîchissement.
- Le tableau de bord personnel (à venir) résumera vos contributions, commentaires et favoris.

## 7. Administration (réservée au staff)
- Les comptes administrateurs accèdent au panneau `/admin/index.php`.
- Fonctions disponibles : suppression d'utilisateurs, de discussions et de réponses, consultation des statistiques globales.
- Chaque action sensible exige une confirmation et repose sur un token CSRF.

## 8. Bonnes pratiques & assistance
- Respectez la charte communautaire disponible dans la section "Liens utiles".
- Signalez tout dysfonctionnement via l'adresse de support listée dans le footer ou via Git.
- Pour une expérience optimale, utilisez un navigateur moderne (Chrome, Firefox, Safari, Edge) mis à jour.

Bonne exploration du Temple !
